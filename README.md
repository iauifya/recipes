# 食譜收集頁

##### 可以收集自己喜歡食譜的小專案，從主頁進入的每一張卡片就代表一個食譜，提供快速查看成分及作法的設計，也可點擊view recipe按鈕查看完整介紹，此外，可手動新增食譜，點擊主頁上方NEW RECIPE按鈕並填入相對應的表格就可完成新增，亦可從卡片右上方的X按鈕刪除食譜
##### 使用vue3開發 + vue cli環境部屬

##### 網址~https://recipes-d9f84.web.app/#/
